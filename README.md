# TechYaka - Geleceğin Kariyer Haritası 🗺️🚀

TechYaka, teknoloji ekosistemindeki fırsatları (Hackathon, Meetup, Staj) harita bazlı sunan, modern ve interaktif bir kariyer keşif platformudur. Bu sürüm, uygulamanın MVP aşamasındaki Frontend mimarisini içermektedir.

## ✨ Öne Çıkan Özellikler (MVP)
- **🌓 Gece / Gündüz Modu (Dark/Light Theme):** Tek tıkla tüm arayüzün (harita katmanları dahil) pürüzsüzce değiştiği dinamik tema.
- **🗺️ İnteraktif Harita & Özel Pinler:** Leaflet JS altyapısı ve temaya uygun tasarlanmış özel SVG işaretçiler (marker).
- **📱 Mobil Öncelikli UI (Bottom Sheet):** Gerçek bir mobil uygulama hissi veren "aşağıdan açılan" detay paneli ve yatay kaydırmalı (swipe) mimari.
- **🔄 Dinamik Durum Yönetimi:** Başvurulan ve kaydedilen etkinliklerin anlık (state-based) olarak arayüzde güncellenmesi ve istatistiklere yansıması.

## 🛠️ Kullanılan Teknolojiler
- **Core:** React, Vite
- **Styling:** Tailwind CSS 
- **Mapping:** React-Leaflet, CartoDB (Voyager & Dark Matter)
