# TechYaka 3 Haftalık Geliştirme Planı

Bu döküman, TechYaka projesinin MVP aşamasına giden 3 haftalık teknik sprint yol haritasını özetler.

* **Hafta 1 (Backend & Mimari Odaklı):** FastAPI ile temel API iskeletinin kurulması, endpoint'lerin (veri uç noktalarının) tanımlanması ve Gemini API entegrasyonu ile ilk veri parse (ayrıştırma) testlerinin yapılması.
* **Hafta 2 (Veri Hattı & Veritabanı):** Farklı kaynaklardan etkinlik verilerinin toplanması, AI ile temizlenmesi ve PostgreSQL/MongoDB veritabanına JSON formatında yapılandırılarak kaydedilmesi.
* **Hafta 3 (Minimal Frontend & Entegrasyon):** React ve Leaflet kütüphanesi kullanılarak minimal bir İstanbul haritasının ayağa kaldırılması. Backend API'den gelen verilerin bu harita üzerinde pin (işaretçi) olarak görselleştirilmesi.
